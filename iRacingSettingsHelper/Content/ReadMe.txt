iRacing Auto Settings
---------------------

An example Configuration file is installed to "My Documents\iRacing Auto Settings\Configuration.xml", edit this configuration to your requirements.

To enable the iRacing Settings Server to connect to the iRacing web front-end, a user-script will need to be installed.

Chrome
------

- Go to Tools/Extensions
- Drag and drop the "iRacing Auto Settings.user.js" file from the "My Documents\iRacing Auto Settings" folder to the Chrome browser
- Click Add to install the user script

Firefox
-------

- Install Grease Monkey
- Drag and drop the "iRacing Auto Settings.user.js" file from the "My Documents\iRacing Auto Settings" folder to the User Scripts add-ons screen
- Follow prompts
